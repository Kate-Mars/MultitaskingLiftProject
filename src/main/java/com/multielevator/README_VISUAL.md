# Multi‑Elevator Visual

### Пассажиры
- Справа есть поле **Passengers to generate (total)** — это *общее* количество пассажиров, которое генератор создаст.
- Значение можно **увеличивать прямо во время работы**: генератор продолжит выпускать людей, пока не достигнет нового лимита.
- Если поставить значение меньше, чем уже сгенерировано, оно будет проигнорировано.

### Скорость и пауза
- Ползунок **Speed** ускоряет/замедляет всю симуляцию (движение, двери, посадка, генерация пассажиров).
- Кнопка **Pause/Resume** ставит симуляцию на паузу.

### Горячие клавиши
- **Space** — пауза/продолжить
- **+ / -** — увеличить/уменьшить скорость
- **G** — ввести новое общее количество пассажиров
